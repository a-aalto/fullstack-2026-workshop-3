About This Workshop

Learning Objectives

In this workshop, you're learning how to build web servers using Express.js, one of the most popular Node.js frameworks.

What You'll Master:

  • Express Basics: Setting up an Express application
  • Middleware: Understanding and using middleware functions
  • Routing: Creating routes with Express Router
  • Static Files: Serving HTML, CSS, and other assets
  • API Endpoints: Building RESTful JSON APIs
  • Error Handling: Implementing custom error handlers

From Workshop 02 to Workshop 03

In Workshop 02, you built a web server using Node.js's built-in HTTP module. Now with Express:

  • Code is much shorter and cleaner
  • Routing is more intuitive
  • No manual file reading for static files
  • JSON responses are simpler
  • Better error handling capabilities
